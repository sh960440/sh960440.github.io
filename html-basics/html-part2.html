Image / Audio / Video Tags

<img/>

用途：加入圖片
屬性：
  • src: 圖片的路徑。可以是目錄中的檔案或是網路上圖片的網址
  • width: 圖片的寬度。若只有指定寬度時則會自動依比例調整高度
  • height: 圖片的高度。若只有指定高度時則會自動依比例調整寬度
  • alt: 圖片的替代文字，用於圖片無法正常顯示時
  • title: 圖片的標題，滑鼠停在圖片上時顯示的文字
範例：從目錄中取得一張圖片，將其高度設置為100px

範例：從網路上取得一張圖片，將其高度設置為100px

<audio>

用途：加入音訊
屬性：
  • src: 音訊的路徑
  • autoplay: 自動播放
  • controls: 顯示控制介面
  • loop: 循環播放
  • muted: 靜音
範例：從目錄中取得音訊檔，顯示控制介面，預設其為靜音，並開啟循環播放

<video>

用途：加入影片
屬性：
  • src: 影片的路徑
  • autoplay: 自動播放
  • controls: 顯示控制介面
  • loop: 循環播放
  • muted: 靜音
  • poster: 使用者播放影片前顯示的圖片
  • width: 影片播放器的寬度
  • height: 影片播放器的高度
範例：從目錄中取得一個影片檔，設置其寬度為200px，顯示控制介面，並設置一張預設圖片

Link Tags

<a>

用途：添加超連結
屬性：
  • href: 指定目標URL
  • download: 指定下載連結
  • title: 指定滑鼠放在連結上時顯示的文字
  • target: 指定打開目標URL的方法，可能的值有
    • _blank: 在新視窗打開
    • _parent: 在父窗口打開
    • _self: 在此視窗打開，_self為預設值
範例：建立一個文字連結，點擊後在新視窗顯示一張韓國國旗，滑鼠放在連結上時會顯示目的地
看看韓國國旗
範例：建立一個圖片連結，點擊後讓此視窗移動至Google首頁，滑鼠放在連結上時會顯示目的地

<link>

用途：定義文檔與外部資源的關係，最常見的用途是連結stylesheet
屬性：
  • href: 指定被連結檔案的位置
  • rel: relationship，定義此檔案與被連結檔案之間的關係，包括alternate、icon、stylesheet等21種可用參數
  • type: 規定被連結的MIME類型

<nav>

用途：定義導航連結
範例：定義到三個網站的導航連結

Frames

<iframe>

用途：指定一個inline frame，inline frame用來在目前的html檔案中嵌入另一個檔案 屬性：
  • height: 定義<iframe>的高度
  • weight: 定義<iframe>的寬度
  • name: 定義<name>的名稱
  • src: 指定在<name>中顯示的檔案的URL
範例：嵌入一個網頁
