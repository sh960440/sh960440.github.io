通用概念

  • 同時有Opening tag跟Closing tag的tag叫做Content(Container) tag
    • 範例：html、body、h1、h2、h3、p
  • 使用時只需要一個tag，不需要分Opening tag跟Closing tag的叫做Non-container tag
    • 範例：image、br
  • HTML的Tag根據所占空間大致分為兩種
    1. Block Level Element: 佔據一整列的空間，接在後面的文本將會自動從下一行開始，例如h1、li、ul、p、table、div
    2. Inline Element: 只佔據文本內容的大小，接在後面的文本將不會自動換行，例如b、em、a、img、input

Basic HTML Tags

!DOCTYPE

用途：定義文檔的類型

html

用途：定義html的內容，在這個tag之間的內容皆為html

head

用途：定義關於文檔的資訊，通常在body tag的上面

title

用途：定義文檔的標題，標題即為瀏覽器分頁上看到的網頁名稱，通常在head tag當中

body

用途：指定文檔的主體，通常在head tag的下面

header

用途：指定文檔的頂部部分，通常在body tag當中，main tag上面

main

用途：指定文檔的主體部分，通常在body tag當中，main tag和footer tag之間

footer

用途：指定文檔的底部部分，通常在body tag當中，main tag下面

br/

用途：換行

hr/

用途：顯示一條水平分割線

p

用途：定義一個段落

h1 / h2 / h3 / h4 / h5 / h6

用途：顯示h1~h6大小的標題

Formatting Tags

b / strong

用途：粗體字。b或stronng都會讓內容的文字變成粗體字，差別在於b是樣式標籤，純粹改變字體樣式，而strong是語意標籤，代表內容具有重要性
範例：將這段文字變成粗體

i / em / var

用途：斜體字。i、em或var都會讓內容的文字變成斜體字，差別在於i是樣式標籤，純粹改變字體樣式，em和var是語意標籤，em有強調文字內容的意義，而var代表變量
範例：將這段文字變成斜體

u / ins

用途：帶底線的文字。u或ins都會讓內容的文字加上底線，差別在於u純粹改變字體樣式，而ins通常與del一起使用，代表刪除舊文本後插入更新的文本
範例：將這段文字加上底線

del / s

用途：帶刪除線的文字。del或s都會讓內容的文字加上刪除線
範例：將這段文字加上刪除線

mark

用途：帶有底色記號的文字
範例：將這段文字加上底色

small

用途：小型字
範例：將這段文字變成小型字

q

用途：定義短引用文字，在文字的兩側加上引號
範例：將這段文字變成引用文字

sub

用途：下標文字
範例：用下標文字表示化學式H2O

sup

用途：上標文字
範例：用上標文字表示2的10次方210

pre

用途：定義預格式化的一段文字，在代碼中，pre元素內的空格或換行都會顯示在網頁上，而字體會呈現等寬體
範例： 
        顯示一段的預格式文本
        不需要使用br/就能換行
        也能保留   空白
註記：和p一樣，pre是一個block level element

progress

用途：顯示進度條
屬性：
  • max: 完成全滿需要的值
  • value: 目前進度的值
範例：顯示半滿的進度條任務進度
