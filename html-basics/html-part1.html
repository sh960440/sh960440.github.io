HTML Basics - 1

通用概念

  • 同時有Opening tag跟Closing tag的tag叫做Content(Container) tag
    • 範例：html、body、h1、h2、h3、p
  • 使用時只需要一個tag，不需要分Opening tag跟Closing tag的叫做Non-container tag
    • 範例：image、br

常用Tag

!DOCTYPE

用途：定義文檔的類型

html

用途：定義html的內容，在這個tag之間的內容皆為html

head

用途：定義關於文檔的資訊，通常在body tag的上面

title

用途：定義文檔的標題，標題即為瀏覽器分頁上看到的網頁名稱，通常在head tag當中

body

用途：指定文檔的主體，通常在head tag的下面

header

用途：指定文檔的頂部部分，通常在body tag當中，main tag上面

main

用途：指定文檔的主體部分，通常在body tag當中，main tag和footer tag之間

footer

用途：指定文檔的底部部分，通常在body tag當中，main tag下面

br/

用途：換行

hr/

用途：顯示一條水平分割線

p

用途：定義一個段落

h1 / h2 / h3 / h4 / h5 / h6

用途：顯示h1~h6大小的標題
